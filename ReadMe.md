# Kitchen Chaos 3D 模拟烹饪游戏（Unity复刻项目）

## 🎮 项目简介
本项目是跟随 CodeMonkey《胡闹厨房》全流程教程，从零开始独立复刻的 3D 模拟烹饪游戏。
教程总时长约 10.5 小时，目前已完成 **90% 核心功能开发**（剩余音效、最终打磨等收尾内容），完整实现了食材切配、烹饪、装盘、订单验证、玩家交互等核心玩法，全面覆盖 Unity 游戏开发全流程。

## 🛠️ 技术栈
- 开发引擎：Unity 2022
- 开发语言：C#
- 核心技术：
  - C# 面向对象编程、接口设计、委托与事件驱动、单例模式
  - Unity 核心模块：UGUI、物理系统、Animator、Character Controller、Cinemachine 相机系统
  - 输入系统：New Input System、键盘/手柄双端输入、自定义按键重绑定
  - 数据驱动：ScriptableObjects 配置化管理食材/配方/切割规则
  - 有限状态机：实现烹饪状态（生肉/煎烤/熟透/烧焦）精准切换
  - 视觉效果：Post Processing 后处理、Shader Graph 自定义 UI/场景效果
  - 其他：世界空间 UI、音效系统、完整游戏流程（开始/结束/暂停/主菜单）
